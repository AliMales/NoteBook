## 监听页面中的请求

```javascript
const originXHR = window.XMLHttpRequest;
const originOpen = window.XMLHttpRequest.prototype.open;
window.XMLHttpRequest = () => {
  var realXHR = new originXHR();
  realXHR.open = (method, url) => {
    originOpen.call(realXHR, method, url);
  };
  return realXHR;
};
```
