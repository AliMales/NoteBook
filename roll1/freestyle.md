**随笔**

```
今夜我在德令哈
夜色笼罩
今夜我只有美丽的戈壁 
这是雨水中一座荒凉的城
姐姐，今夜我不关心人类
我只想你
```

赏析海子的日记，这是我钟情的片段。可惜这位诗人自杀了！

海子，原名查海生，
1964年3月26日生于安徽省怀宁县高河镇查湾村
1989年3月26日于山海关与龙家营之间的火车慢行道上卧轨自杀

他的朋友曾在书中提到海子自杀的细节

```
中午大概到的山海关，下午在山海关呆了一下午。心中有再大难处的人，经过震耳欲聋的轰轰火车，在看大自然，心中的闷自然化解了。看来海子是抱着自杀的决心了。下了火车，一个人在空旷的平原徘徊，春天也没有留住他。
```
